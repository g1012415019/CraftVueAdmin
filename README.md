## 🚀 简介

`Naive Ui Admin` 是一款 完全免费 且可商用的中后台解决方案，基于 🌟 `Vue3.0` 🌟、🚀 `Vite` 🚀、✨ [Naive UI](https://www.naiveui.com/) ✨ 和 🎉 `TypeScript` 🎉。
它融合了最新的前端技术栈，提炼了典型的业务模型和页面，包括二次封装组件、动态菜单、权限校验等功能，助力快速搭建企业级中后台项目

## 🌈 特性
📦 二次封装的实用高扩展性组件
🎨 响应式、多主题、多配置，快速集成，开箱即用
🚀 强大的鉴权系统，支持 三种鉴权模式，满足多样业务需求
🌐 持续更新的实用性页面模板和交互设计，简化页面构建
📋 **全新低代码表格组件**：支持拖拽排序、复选框批量操作、悬浮操作栏等高级功能

## 📋 表格组件特性

### 🎯 核心功能
- ✅ **拖拽排序**：支持行拖拽重新排序，带视觉反馈
- ✅ **批量操作**：复选框选中后显示操作栏
- ✅ **悬浮操作**：选中行时显示固定操作栏
- ✅ **内联编辑**：双击行进入编辑模式
- ✅ **右键菜单**：丰富的上下文菜单操作
- ✅ **响应式设计**：自适应不同屏幕尺寸

### 🛠️ 配置化特性
- 📊 **动态列配置**：可视化配置表格列
- 🎨 **主题定制**：支持多种主题和样式配置
- 📱 **移动端适配**：完美支持移动设备
- 🔧 **插件扩展**：支持自定义插件和扩展

## 🏗️ 使用

- 获取项目代码

```bash
git clone https://github.com/jekip/naive-ui-admin.git
```

- 安装依赖

```bash
cd naive-ui-admin

pnpm install

```

- 运行

```bash
pnpm run dev
```

- 打包

```bash
pnpm build
```

## 📋 Git 贡献提交规范

- 参考 [vue](https://github.com/vuejs/vue/blob/dev/.github/COMMIT_CONVENTION.md) 规范 ([Angular](https://github.com/conventional-changelog/conventional-changelog/tree/master/packages/conventional-changelog-angular))

  - `feat` 增加新功能
  - `fix` 修复问题/BUG
  - `style` 代码风格相关无影响运行结果的
  - `perf` 优化/性能提升
  - `refactor` 重构
  - `revert` 撤销修改
  - `test` 测试相关
  - `docs` 文档/注释
  - `chore` 依赖更新/脚手架配置修改等
  - `workflow` 工作流改进
  - `ci` 持续集成
  - `types` 类型定义文件更改
  - `wip` 开发中
